# MiniPLInterpreter

## Running the program
In the project root folder, run:
`Cargo run path_to_file.txt` (e.g. cargo run sample1.txt)
to parse a single file.


`Cargo run` to open a command line parser where you can type code and 
hit enter to parse it.
